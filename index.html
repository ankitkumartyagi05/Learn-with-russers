What is a Variable?

Variable ek container hota hai jisme hum data store karte hain.

Imagine karo ek dabba jisme tum apni cheezein rakh sakte ho. Us dabbe mein tum "Pen", "Phone", ya "Money" rakh sakte ho.

Example Code
my_name = "Rohit"
age = 15
print(my_name)
💡
Yaad Rakhna: Variable naam hamesha meaningful hona chahiye taaki code padhne mein aasani ho.
