# 🚀 Learn_with_Russers

### Gamified AI Learning Platform (Beginner to Pro)

> “Learning ko game bana do — aur students khud daily wapas aayenge.”

---

## 🎯 Project Overview

**Learn_with_Russers** ek AI-powered, gamified learning platform hai jo specially beginners aur weak students ke liye design kiya gaya hai.

Iska main goal hai:

* Learning ko simple banana (Hinglish me)
* Students ko daily engage rakhna (game system ke through)
* AI ke help se personalized growth dena

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/learn_with_russers.git
cd learn_with_russers
```

### 2️⃣ Install Dependencies

#### Frontend:

```bash
cd frontend
npm install
```

#### Backend:

```bash
cd backend
npm install
```

---

## 🔐 Environment Variables

Create `.env` file in backend:

```
PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_api_key
```

---

## ▶️ Run Project

### Backend:

```bash
npm run dev
```

### Frontend:

```bash
npm start
```
